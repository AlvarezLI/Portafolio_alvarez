Proyecto 1: Modelo de comestibilidad de hongos

Este proyecto es parte de mi proyecto final para un curso de ciencia de datos, donde clasifiqué diferentes hongos según su comestibilidad.

Fuente del conjunto de datos

El dataset utilizado es el Mushroom Classification dataset de Kaggle, que provee una descripción completa de las características de los hongos.

Técnicas de Preprocesamiento de Datos

  • Verificación de Valores Nulos: Se gestionaron valores faltantes para asegurar la integridad del dataset.
  • Codificación de Variables Categóricas: Transformación de características categóricas en valores numéricos para entrenamiento de modelos.

Algoritmos de Machine Learning

  • Random Forest Classifier
  • Support Vector Machine (SVM)
  • Logistic Regression

Proyecto 2: Modelo de predicción del tiempo

En este proyecto, realicé un análisis profundo de un dataset sintético diseñado para clasificar tipos de clima.

Técnicas de Preprocesamiento de Datos

  • Manejo e Imputación de Valores Nulos: Se gestionaron valores faltantes para mantener la calidad del dataset.
  • Codificación de Variables Categóricas: Se convirtieron variables categóricas en un formato adecuado para el entrenamiento de modelos.
  • Eliminación de Outliers: Se utilizó el método IQR para eliminar valores atípicos.

Algoritmos de Machine Learning

  • Random Forest Classifier
  • Gradient Boosting Machine (GBM)
  • Logistic Regression
  • Decision Tree Classifier

Optimización y Evaluación de Modelos

  • Precisión: Medición de la exactitud general de las predicciones.
  • Matriz de Confusión: Evaluación del rendimiento de los modelos de clasificación.

Se compararon los modelos para identificar el mejor ajuste y se realizó optimización de hiperparámetros para mejorar los resultados.
