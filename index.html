Proyecto 1: Modelo de Edibilidad de Hongos

Este proyecto es parte de mi proyecto final de un curso de Data Science, donde clasifiqué diferentes tipos de hongos según su edibilidad.

Fuente del Conjunto de Datos

El conjunto de datos utilizado es el dataset de clasificación de hongos de Kaggle, que proporciona una descripción detallada de las características de los hongos.

Técnicas de Preprocesamiento de Datos

  • Comprobación de Valores Nulos: Se aseguró la integridad del conjunto de datos mediante un manejo adecuado de los valores faltantes.
  • Codificación de Variables Categóricas: Se transformaron las características categóricas en valores numéricos para un entrenamiento efectivo del modelo.

Algoritmos de Aprendizaje Automático

  • Clasificador Random Forest
  • Máquina de Soporte Vectorial (SVM)
  • Regresión Logística

Proyecto 2: Modelo de Predicción Climática

En este proyecto, realicé un análisis exhaustivo de un conjunto de datos sintético diseñado para clasificar diferentes tipos de clima.

Técnicas de Preprocesamiento de Datos

  • Manejo y Imputación de Valores Faltantes: Se abordaron los valores faltantes para mantener la calidad del conjunto de datos.
  • Codificación de Variables Categóricas: Se transformaron las variables categóricas en un formato apropiado para el entrenamiento del modelo.
  • Eliminación de Valores Atípicos: Se aplicó el método del rango intercuartílico (IQR) para eliminar los valores atípicos.

Algoritmos de Aprendizaje Automático

  • Clasificador Random Forest
  • Gradient Boosting Machine (GBM)
  • Regresión Logística
  • Clasificador Árbol de Decisión

Optimización y Evaluación del Modelo

  • Precisión: Se midió la corrección general de las predicciones.
  • Matriz de Confusión: Se evaluó el rendimiento de los modelos de clasificación.

Se compararon los modelos para identificar el mejor ajuste y se realizó una optimización de los hiperparámetros para mejorar los resultados.

Proyecto 3: Análisis de Obras Públicas en Excel y Tableau

En este proyecto, realicé un análisis exhaustivo de datos de obras públicas en Argentina para obtener información sobre la distribución y el estado de las inversiones en infraestructura a nivel regional, utilizando Excel y Tableau para optimizar la visualización y análisis de los datos.

Fuente del Conjunto de Datos

El conjunto de datos proviene del Mapa de Inversiones del Ministerio de Obras Públicas de Argentina, el cual proporciona información detallada sobre las obras públicas en diversas provincias y su estado de avance.

Técnicas de Análisis y Preprocesamiento de Datos

  • Limpieza y Validación de Datos: Se eliminaron errores y se validaron los valores, tanto cualitativos como cuantitativos, para garantizar la precisión del análisis.
  • Tablas Dinámicas: Se usaron tablas dinámicas en Excel para facilitar la exploración de los datos y responder preguntas clave como el número de obras por tipo, estado y ubicación geográfica.
  • Segmentación de Datos: Se agregaron segmentadores para permitir la segmentación dinámica de los datos por categorías como tipo de obra, año, estado de ejecución y provincia.

Visualización y Tableros

Para mejorar la interpretación de los datos, se crearon tableros interactivos tanto en Excel como en Tableau:

  • Tablero en Excel: Se diseñó un tablero interactivo con gráficos y segmentadores para visualizar rápidamente la distribución y el estado de las obras públicas.
  • Tablero en Tableau: Se creó una visualización avanzada para facilitar la exploración visual de los datos y permitir el filtrado y segmentación de las obras según diversas variables de interés.

Hallazgos Clave

Este análisis proporcionó una visión general del estado de las obras públicas en Argentina, destacando las regiones con mayor y menor inversión, así como los tipos de proyectos más comunes y su estado de avance.

Proyecto 4: Festivales Populares en Argentina

Este proyecto se centra en el análisis de eventos y festivales populares en Argentina a través de un informe desarrollado en Power BI. El objetivo es explorar la asistencia a eventos, la ubicación y los temas de los mismos.

Fuente del Conjunto de Datos

El conjunto de datos utilizado es una recopilación de información sobre eventos y festivales populares en Argentina, incluyendo detalles como el número de asistentes, la ubicación de los eventos y los temas de los mismos.

Técnicas de Preprocesamiento de Datos

  • Control de Valores Nulos: Se manejaron los valores faltantes para asegurar la integridad del conjunto de datos.
  • Codificación de Variables Categóricas: Se transformaron las características categóricas para su análisis de datos.

Visualización y Análisis

  • Asistencia: Gráficos que muestran el número de asistentes por mes, año y tipo de entrada.
  • Ubicación de los Eventos: Un mapa muestra la distribución de los eventos por provincia.
  • Análisis de Temas: Clasificación de eventos por tema y tipo de entrada.

Resultados Clave

Este análisis proporciona una visión integral de la participación en festivales en Argentina, destacando tendencias en asistencia, distribución de eventos y temas predominantes.

Proyecto 5: Análisis de Datos Musicales de Spotify

Este proyecto se centra en el análisis de datos de canciones de Spotify para explorar las características musicales y su relación con la popularidad de las canciones.

Fuente del Conjunto de Datos

El conjunto de datos utilizado fue descargado de Kaggle, el cual proporciona información sobre características de las canciones como tempo, valencia, energía y popularidad.

Técnicas de Preprocesamiento de Datos

  • Extracción de Datos: Se extrajeron los datos relevantes de las canciones para su análisis.
  • Visualización de Datos: Se exploraron las distribuciones de las características musicales y su relación con la popularidad.

Visualización y Análisis

  • Distribución de Tempo: Un histograma muestra la distribución del tempo entre las canciones.
  • Energía vs Popularidad: Un diagrama de dispersión ilustra la relación entre la energía de la canción y su popularidad.

Resultados Clave

Este análisis proporciona información sobre cómo las características musicales influyen en la popularidad de las canciones en Spotify, revelando tendencias a través de diferentes géneros.
